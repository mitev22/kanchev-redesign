# 05 — Broadsheet (редакционен минимализъм)

**Pitch:** Практиката като първа страница на добър вестник — типографска честност вместо брошура: хартия, мастило, една вермилионова нишка.

## Mood

Literate, calm authority. Никакви снимки, никакви сенки, никакви заоблени ъгли — само Literata върху хартиено-бял фон, 1px хартиени линии и дисциплиниран ритъм. Стремежът: непознат да си помисли „сериозно, честно, никакъв маркетинг" за 3 секунди.

## Target

Дизайн-грамотни възрастни; скептици към „маркетингова" здравна комуникация; пациенти, които избират лекар по репутация и тон, не по стокови фотографии. Съзнателен контраст с тъмния елегантен editorial-драфт на практиката: това направление е СВЕТЛА хартия.

## Key design moves

1. **Masthead** — 3px горен кант (на `<body>`), центриран wordmark в Literata, навигация като small-caps ред между два хеърлайна (вестникарско folio).
2. **Pure-type hero** — огромно заглавие (clamp 2.3–4.2rem, една дума в курсив) + standfirst; нула образност.
3. **Услуги като ИНДЕКС** — редове „01–06" като съдържание на вестник: вермилионов номерал / име / описание / →, разделени с хеърлайни; на началната страница редовете са линкове към `services.html#s01…#s06`.
4. **Drop cap** — `::first-letter` на водещия абзац в „За доктора" (есеен отвор).
5. **Pull-quotes** — отзиви като големи курсивни цитати с оувърсайзд висяща „ (ghost, 4.25rem), атрибуция в Inter 11px uppercase; на десктоп три вестникарски колони с вертикални хеърлайни.
6. **Табулирани факти** — работно време / прием / НЗОК като definition-таблици с хеърлайни (контакти, за доктора, странична колона на услуги).

Десктоп асиметрия: `.split` = широка текстова колона + 19rem мета-колона (услуги: consult бележка + факти; за доктора: портретна бележка + запазен час; контакти: карта-бокс).

## Token summary

| Token | Value | Note |
|---|---|---|
| `--color-paper` | `#FAFAF6` | фон |
| `--color-ink` | `#161616` | текст, плътни бутони (17.3:1 върху paper) |
| `--color-gray` | `#63635E` | втори текст (5.76:1 — AA ✓) |
| `--color-accent` | `#C0361B` | vermilion — САМО линкове, номерали, CTA (5.3:1 — AA ✓) |
| `--line` | `#DEDED6` | 1px хеърлайни |
| `--font-display/body` | Literata (opsz 7..72) | 400/500/600 + italic 400 |
| `--font-meta` | Inter 400/500 | само 11px uppercase .14em мета-етикети |
| `--radius-none` | `0` | без радиуси, без сенки, без градиенти |
| `--space-1…9` | 0.25–6rem | ритъм |
| `--measure` | 65ch | максимална дължина на ред |

AA бележка: палитрата мина проверка без корекции — gray и accent са и двете над 4.5:1 върху paper, така че body копи, мета-етикети и линкове са съответстващи както са зададени.

## Component inventory

`.masthead` (meta row / brand / nav) · `.hero` (eyebrow, title, standfirst, rating, actions) · `.stats` (4 колони с хеърлайни) · `.index` / `.index__row` (услуги-съдържание; линк на Home, `<article>` + h2 на Services) · `.steps` (01–03) · `.quote` / `.quotes` (pull-quotes + `.quotes__source`) · `.consult` (rule-bounded банер) · `.facts` (`--stack` вариант за тясната колона) · `.split` (асиметричен лейаут) · `.prose` + `.dropcap` · `.creds` (номериран лист с decimal-leading-zero броене) · `.note-box` (портрет/карта плейсхолдъри) · `.faq` (`<details>` ruled list с +/−) · `.benefits` · `.form` (underline инпути, custom select стрелка, квадратни checkbox-и) · `.notice` (спешност) · `.cta-band` · `.colophon` · `.action-bar` (мобилен фиксиран бар ≤720px) · `.btn` / `.link-tel` / `.meta-label` / `.skip-link`.

## Photo-free by design

**Deliberately photo-free — typography is the identity.** Портретът и картата са тънко-рамкирани бележки-плейсхолдъри; ако практиката някога поиска снимки, те се вкарват само в тези два слота, без да пипат системата.

## Porting notes

Чист статичен HTML/CSS, file:// съвместим; единствената външна зависимост е Google Fonts (Literata + Inter, кирилица). JS е само demo form handler на contact.html (навигацията wrap-ва без JS) — за Phase 2 се закача Web3Forms ключът от коментара над формата и се маха handler-ът.
